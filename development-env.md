# 搭建开发环境

### 概览

1. 开发语言
   - HTML
   - SCSS
   - JavaScript
   - TypeScript
   - Vue 3
2. 构建工具
   - Vite
3. 包管理工具
   - pnpm
4. 规范化工具
   - ESLint：JavaScript 代码检查工具。
   - Prettier：代码格式化工具。
   - Stylelint：样式规范检查工具。
   - Husky：设置 Git hooks 验证 Git 提交的工具。
   - @commitlint/cli + @commitlint/config-conventional：Git 提交信息的规范工具。
5. 单元测试
   - Vitest
6. 文档
   - Vitepress
7. 代码部署
   - GitHub
8. 持续集成 CI
   - Github Action
9. package 发布
   - npm
